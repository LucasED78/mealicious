# How to install

1 - Clone this project
```bash
git clone https://github.com/lucased78/mealicious
```
2 - Install the dependencies
```bash
cd mealicious && npm install
```

3 - Run the robot
```bash
npm run dev
```

# Running the tests
To run the tests, simply run the following command:
```bash
npm run test
```
